# Portfolio

Next.js + Rsbuild TypeScript portfolio showcasing 3D graphics and WebAssembly.

## Development

- `npm run dev` – start development server
- `npm run build` – build for production
- `npm test` – run tests (none yet)

## Features

- 3D cylinder hero component with rotating cards
- Responsive brutalist design inspired by dithering techniques
- SSR and SSG examples
- Minimal WebAssembly demo loaded from base64
